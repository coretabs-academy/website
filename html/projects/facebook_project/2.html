تقسيم ملفات المشروع

في البدأية يجب أن نتفق على تقسيم ملفات المشروع حتى يكون عملنا مرتب.

سنحتاج إلى التالي :

-مجلد المشروع الرئيسي وسأقوم بتسميتة facebook

بداخل هذا المجلد سنقوم بوضع التالي:

1- مجلد images والذي سنضع فيه جميع الصور التي سنستخدمها في المشروع

2- ملف مستند الـ HTML وسأقوم بتسميته index.html

3- مجلد css والذي سنضع فيه ملفات الـ css

4- ملف style.css بداخل مجلد css كما هو موضح بالصورة أدناه

بعد إنتهائك من إنشاء ملف المشروع قم بالتالي:

1- قم بتجهيز البنية الأساسية لملف index.html

2- قم بربط ملف index.html بملف التنسيقات style.css بإستخدام

<link rel="stylesheet" href="css/style.css" />
