1- سأقوم بإعطاء الجدول عرض بقيمة 400px
2- سأقوم بتغيير نوع الخط حتى يطابق الموقع الأصلي

البدء بالعمل على التنسيقات

تنسيق الحقول:

سأبدا بإعطاء كل حقول الإخال عرض بقيمة 100% كالتالي:

.float-right table input[type=text],
.float-right table input[type=email],
.float-right table input[type=password]{
width: 100%;
}

النتيجة:

حصلنا على المطلوب بجعل الحقول تتمدد بعرض 100% من حجم الخلية ولكن ظهرت لنا مشكلة حيث أن حقل الإسم الأول ظهر أصغر من حقل الإسم الأخير بينما نحن نريدهم أن يكونوا بنفس العرض لهذا حتى نحل هذا المشكلة سنقوم بإعطاء الخلية الواحدة عرض بمقدار 50% وذلك لانه لدينا خليتين في كل صف:

.float-right table td{
/*border:1px solid #000;*/
padding: 4px; /*قمت كذلك بالتعديل على الهوامش*/
width: 50%;
}

النتيجة:

لنكمل العمل على تنسيق الحقول:
1- سأقوم بإضافة حدود للحقول بلون محدد
2- سأجعل زوايا الحقول دائرية بقيمة 5px
3- سأقوم بإعطاء الحقول هوامش داخلية
4- سنجعل حجم الخط 18px
5- سأقوم بجعل العنصر يحتسب الهوامش من حجمة الأصلي دون أي زيادة أخرى

.float-right table input[type=text],
.float-right table input[type=email],
.float-right table input[type=password]{
width: 100%;
border: 1px solid #bdc7d8;
border-radius: 5px;
padding: 8px 10px;
font-size: 18px;
box-sizing: border-box;
}

إذا لنعود لإضافة هذه الصفة للعناصر - سأقوم بشرح إضافتها لإحد عناصر حقول الإدخال وأنت اكمل الباقي كالتالي:
