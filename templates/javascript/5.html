الجزء الثاني من Data types

في الدرس السابق تعرفنا على الـ numbers و الـ strings. في هذا الدرس سنكمل مناقشة أنواع البيانات Data Type الأخرى.

النوع الثالث: undefined

عند قيامك بتعريف متغير بدون إسناد أي قيمة إلية تقوم الجافاسكربت بإسناد القيمة undefined لذلك المتغير (ما يعني ان قيمته غير محددة).

مثال:

var x;

لنقوم بطباعة قيمة المتغير x على الـ console ونشاهد القيمة التي تشير اليها x

var x;
console.log("x value = " + x);

النتيجة:

اذا كانت لا تعرف كيف تستخدم الـ console راجع الدرس الأول.

النوع الرابع: null

القيمة null تعني لا شئ وهي قيمة تسند للمتغير في حال أردت جعلة بدون أي قيمة

مثال:

var x = null;

لنقوم بطباعة قيمة المتغير x على الـ console ونشاهد القيمة التي تم تخزينها في x

ربما تتسأل ما الفرق بين undefined و null ؟

في الواقع undefined و null متساويات في القيمة ومحتلفات بالنوع وسنثبت ذلك في الدرس الخاص بعمليات المقارنة.

  • نستطيع القول بأن null قيمة تقوم انت بإسنادها للمتغير لجعلة بدون قيمة بينما undefined تقوم الجافاسكربت بإسنادها للمتغير في حال انك قمت بتعريفة وتركته بدون أي قيمة.
var x = null; // no value
var y; // undefined
  • نستخدم null لمسح قيمة المتغير لنفترض على سبيل المثال قمت بتعريف متغير x واسندت إلية قيمة وثم في وقت لاحق أردت أن تقوم بإسناد لا شئ للمتغير x
var x = 5;
x= null;

هذا ما يكفي لمعرفته بخصوص هذين النوعين في الوقت الحالي حتى لا ترتبك لذا ننتقل للنوع الأخر

النوع الخامس: boolean

هذا النوع يسمى القيم المنطقية وتحتمل قيمتين فقط و هما true او false هذا النوع من البيانات تقوم بالتفاعل معه في حياتك اليومية لنأحذ على سبيل المثال الأمثلة التالية:

ربما قبل قليل قمت بإعطاء شخص إعجاب على موضوع في المنتدى هل لاحظت بأن هذا الزر لا يحتمل سوى حالتين (true / false)

هل سبق لك وأن شاهدت مثل هذا المفتاح ؟

اذا سألتك وقلت لك هل المفتاح هذا الأن في وضع الـ ON ام OFF؟
انا متأكد بأنك ستقول بأنه في وضع ON

ما الذي جعلك تعتقد ذلك ؟
بالتأكيد الذي جعلك تعتقد ذلك هي الرموز المرسومة على المفتاح فهناك رمزين أو بالأصح رقمين 1 و 0

هل تستطيع أن الأن أن تربط علاقة بين الـ 1 و الـ true و بين الـ 0 و الـ false ؟
نستطيع أن نستنتج بأن true تساوي 1 في القيمة وكذلك false تساوي 0 في القيمة, سنقوم بإثبات ذلك في الامثلة القادمة بالدرس التالي.

طريقة إسناد boolean data type للمتغيرات

تستطيع إسناد القيمة true او false للمتغير مباشرة كالتالي:

var lightSwitch = false;
var lightSwitch = true;

مثال:
سأقوم بتعريف متغير واسناد القيمة false الية ومن ثم اظهر قيمته في نافذة منبثقة 

<script>
   var lightSwitch = false;
   alert (lightSwitch);

</script>

النتيجة:

الان لنقوم بتعديل الكود السابق بإضافة علامة تعجب ! قبل اسم المتغير بداخل دالة alert:

<script>
   var lightSwitch = false;
   alert (!lightSwitch); // اضفت هنا علامة التعجب ! قبل المتغير

</script>

النتيجة:

هل لاحظت كيف تغيرت قيمة المتغير من false الى true وكانه كان هناك مفتاح في حالة flase و من ثم قمت لاحقاً بتفعيلة true.

بإمكانك اعتبار بأن ما قمنا به بشكل مشابة لما يحصل في الصورة التالية:

ما الذي تعنية علامة التعجب ! في البرمجة ؟
تعني not فعندما وضعنا علامة التعجب امام اسم المتغير الذي يحمل قيمة false كأننا نقول not false

اذا قلت لك not false ماذا يعني ذلك ؟
يعني true اليس كذلك ؟
