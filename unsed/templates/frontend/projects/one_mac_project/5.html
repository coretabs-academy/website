العمل على about-section

في هذا الجزء سنكمل شرح درس تحدي اليوم الثاني.

الخطة كالتالي:

إنتهيناء من العمل على قسم banner الأن سنبدأ في القسم الجديد about-section وسنضعة بشكل منفصل عن القسم السابق كالتالي:

<!--Start banner section-->
<div class="banner">

	<div class="container">
		<h1>مرحبا بكم في مبادرة المليون مبرمج عربي</h1>
		<h3>مبادرة صاحب السمو الشيخ محمد بن راشد آل مكتوم للنهوض بالمجتمعات العربية عبر تمكينهم من لغة المستقبل</h3>
		<div>
			<a href="#" class="join-us">سجل اهتمامك!</a>
		</div>
	</div>

</div>
<!--End banner section-->

<!--Start about section-->
<div class="about-section">
         <!--هنا سيكون عملنا-->
</div>
<!--End about section-->

سنقوم بوضع الـ container كالعادة وكما وضحنا بالدرس السابق

<div class="about-section">
	<div class="container">

	</div>
</div>

سنبدأ بعمل على عنوان هذا الجزء (section) بإستخدام وسم العناوين h2 كالتالي

<div class="about-section">
	<div class="container">
		<h2>عن المبادرة</h2>
	</div>
</div>
ربما تتسأل متى تستخدم h1 او h2 ؟

سنخوض في هذه المواضيع عندما نشرح الـ semantic HTML حيث وان هناك شروط تحسن من موقعك في محركات البحث ما يسمى بالـ SEO بحيث يصبح محتوى موقعك مرتب ترتيب منطقي يسهل على محرك البحث قرأته.

النتيجة:

الأن سنقوم بإضافة التنسيقات التالية للـ class بإسم about-section :

  • 1- هوامش داخلية من اعلى واسفل بقيمة 70px.
  • 2- توسيط النص
.about-section{
	padding: 70px 0;
	text-align: center;
}

النتيجة:

الأن سنضيف التنسيقات التالية للعنوان h2 بداخل القسم about-section:

  • 1- نحتاج تغيير حجم الخط الى 35px
  • 2- نحتاج تغيير لون الخط للون المحدد
.about-section h2{
	font-size: 35px;
	color: #333;
}

النتيجة:

الأن نريد إضافة الخط الظاهر تحت العنوان كما بالصورة التالية:


تستطيع إضافة هذا العنصر بطرق كثيرة وكما قام العديد منكم في التحدي الخاص بهذا الدرس ولكننا نريد أن نتعلم طريقة جديدة أكثر ديناميكية فكما تلاحظ فقد تم إعتماد العنصر h2 في الموقع كالعناوين الرئيسية لكل قسم فاذا ما إستطعنا القيام بهذا التنسيق بإستخدام الـ css فسيكون رائع ولن نحتاج إلى تكرار كود إضافة هذا العنصر كل مرة نريد فيها إضافة عنوان لقسم رئيسي.

ما الخطة؟
سنقوم بإضافة class للعنصر h2 حتى لا يتم اعتماد هذا التنسيق على كل العناصر من نوع h2 فربما في المستقبل قد نحتاج لإضافة عنصر h2 دون ان يتم إضافة هذا الخط تحته او قد تحتاج إضافة هذا التنسيق تحت عنوان أخر كـ h3, سأقوم بتسمية الـclass بإسم underline كالتالي: 

<div class="about-section">
	<div class="container">
		<h2>عن المبادرة</h2>
	</div>
</div>

العمل بـ CSS pseudo-element

الأن سنتعلم شي جديد يسمى CSS pseudo-element هذه العناصر تسمع لك بتعمل تنسيقات على أجزاء محددة من العنصر المستهدف على سبيل المثال عمل تنسيق مختلف لأول حرف, إضافة محتوى قبل (before) او بعد (after) العنصر… الخ. قم بالبحث عنها وتجربتها فالعمل بها ممتع.

العنصر الذي سنحتاج لإستخدامة للوصول إلى النتيجة التي نرغب بها بإضافة الخط تحت العنوان هو after وذلك لأننا نريد إضافة الخط بعد العنوان وطريقة إستخدام هذه العناصر لإستهداف العنصر الذي نرغب بتطبيق التنسيقات المحددة عليه هو كالتالي:

.underline::after{

}
مثال لتوضيح الفكرة

قم بكتابة الخاصية content واعطيها القيمة التي تريد لنفتض نص معين كالتالي:

.underline::after{
	content: " تجريبي";
}

شاهد النتيجة كيف تم إضافة هذه الكلمة تجريبي بعد الكلمة التي كتبناها بداخل عنصر العنوان h2

لاحظ أضفت ::after بعد إسم الـ class الخاص بالعنصر الذي سنستهدفة والأن كل ما علينا فعلة هو عمل التنسيقات اللازمة لإضافة ذلك الخط كالتالي:

  • 1- في البدأية يجب عليك أن تحدد المحتوى الذي تريد إضافته بعد العنصر المستهدف في حالتنا نريد المحتوى ان يكون فارغ لهذا لن نكتب شئ بدأخلة كالتالي:
.underline::after{
	content: "";
}
  • 2- أريد أن يكون العنصر الجديد من نوع block حتى ينزل لسطر جديد ونستطيع التحكم بإبعادة
.underline::after{
	content: "";
    display: block;
}
  • 3- سأقوم بتحديد عرضة بـ 100px وإرتفاعة بـ 2px
.underline::after{
	content: "";
    display: block;
    width: 100px;
    height: 2px;
}
  • 4- سأقوم بتغيير لون الخلفية لهذا العنصر للون المطابق للموقع الأصلي
.underline::after{
	content: "";
    display: block;
    width: 100px;
    height: 2px;
	background-color:#11b7e6;
}

النتيجة:


رائع! حصلنا على الشكل المطلوب تبقى لنا أن نقوم بتوسيطة بإستخدام الهوامش الخارجية وعطائة هوامش من اعلى بقيمة 4px واسفل بقيمة 40px كما سبق وشرحنا بهذا الدرس من هنا. 

.underline::after{
	content: "";
    display: block;
    width: 100px;
    height: 2px;
	background-color:#11b7e6;
	margin: 4px auto 40px;
}

النتيجة:

الأن لنقوم ببعض التنسيقات على القائمة ul كالتالي:

  • 1- أريد لعناصر القائمة ان تكتب من اليمين الى اليسار لهذا سأعطيها محاذاة لليمين
  • 2- نريد أن نقوم بتغيير نوع النقاط التي تظهر على يمين كل عنصر في القائمة الى الشكل النوع المستخدم في الموقع الأصلي
.info-list ul {
	text-align: right; /*محاذاة النص لليمين*/
	list-style-type: circle; /*تغيير نوع النقاط*/
}

النتيجة:

الان لنستهدف كل عنصر li من هذه القائمة ونعطية التنسيقات التالية:

  • 1- لنقوم بإعطائها هامش سفلي بقيمة 18px
  • 2- لنقوم بتغيير حجم الخط الى 14px
  • 3- لنقوم بتغيير لون الخط ليطابق الموقع الاصلي
.info-list ul li{
	margin-bottom: 18px;
	font-size: 14px;
	color: #333;
}

	text-decoration: none;
	color: #00aaf6;
