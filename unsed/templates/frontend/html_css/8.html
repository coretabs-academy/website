أساسيات CSS

في الدرس السابق تعرفنا على طرق إضافة أكواد الـ CSS إلى مستند الـ HTML, في هذا الدرس سنتكلم عن أهم الأساسيات لكتابة أكواد الـCSS.

الكود في الصورة أعلاة هو مثال لأبسط المكونات الأساسية لعمل تنسيق CSS لعناصر الـ HTML وسنفصلها كالتالي:

1- الـ Selector او ما يمكننا تسميته العنصر المستهدف:

كما قمنا بالتوضيح سابقاً تحتاج الـ CSS لمعرفة من هو العنصر المستهدف الذي ستقوم بتطبيق التنسيقات المرغوبة عليه وسنذكر أبرز ثلاث طرق لعمل ذلك:

  • الطريقة الأولى: Element Selector
    في هذه الطريقة تقوم بإستهداف عنصر بإستخدام الوسم ذاته كما بالصورة السابقة قمنا بإستهداف جميع عناصر الوسم <p>.

لنأخذ مثال:

              <!--HTML CODE-->
<p>Hello world!</p>
<p>One Million Arab Coder</p>
<!-- CSS CODE -->
<style>
  p {
    color:red;
  }
</style>

قم بتطبيق المثال السابق.

النتيجة كالتالي:

الكود كاملاً:

                 <!doctype html>
<html>
 <head>
   <meta charset="utf-8">
   <title>One Million Arab Coder</title>
   <style>
     p{
       color:red;
     }
   </style>
 </head>
 <body>
   <p>Hello world!</p>
   <p>One Million Arab Coder</p>
 </body>
</html>

- لاحظ انه تم إستهداف جميع عناصر الوسم p دون أي إستثناء
- لاحظ عند تعريفك للـ Element Selector في الـ CSS تقوم بكتابة إسم الوسم بدون أي إضافات أخرى

  • الطريقة الثانية: ID Selector
    قمنا بشرح فكرة الـ ID في الدرس السابق وكما لاحظت أننا نقوم بتعريفة من أجل إستهداف عنصر واحد فريد من نوعة في مستند الـ HTML و يعتبر واحد من أهم الـ Selectors وأكثرها إستخداماً.

لنأخذ مثال:

              <!--HTML CODE-->
<p id="paragraph">Hello world!</p>
<p>One Million Arab Coder</p>
<!-- CSS CODE -->
<style>
  #paragraph{
    color:red;
  }
</style>

- لاحظ انه تم إستهداف العنصر الذي يحمل الـ id المحدد فقط دون سواه.
- لاحظ عند تعريفك للـ id Selector في الـ CSS تقوم بكتابة # ثم تكتب إسم الـ id كما في مثالنا أعلاة #paragraph.

  • الطريقة الثالة: Class Selector
    قمنا بشرح فكرة الـ Class في الدرس السابق وكما لاحظت أننا نقوم بتعريفة من أجل إستهداف عنصر او مجموعة من العناصر في مستند الـ HTML و يعتبر واحد من أهم الـ Selectors وأكثرها إستخداماً.

لنأخذ مثال:

<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>One Million Arab Coder</title>
    <style>
      #paragraph{
        color:red;
      }
    </style>
  </head>
  <body>
    <p id="paragraph">Hello world!</p>
    <p>One Million Arab Coder</p>
  </body>
</html>

قم بتطبيق المثال السابق.

النتيجة كالتالي:

الكود كاملاً:

<!doctype html>
<html>
  <head>
    <meta charset="utf-8">
    <title>One Million Arab Coder</title>
    <style>
      .paragraph{
        color:red;
      }
    </style>
  </head>
  <body>
    <p class="paragraph">Hello world!</p>
    <p class="paragraph">One Million Arab Coder</p>
    <p>I don't have a class</p>

- لاحظ انه تم إستهداف عنصرين يملكان نفس إسم الكلاس paragraph
- لاحظ عند تعريفك للـ Class Selector في الـ CSS تقوم بكتابة . ثم تكتب إسم الـ class كما في مثالنا أعلاة .paragraph

2- الـ Porperty او الخاصية:

الخطوة التالية بعد أن تقوم بتحديد العنصر المستهدف بإستخدام أحد الطرق أعلاة هو أن تبدأ بإضافة التنسيقات له, وللقيام بذلك يجب أن تقوم بتحديد نوع الخاصية المراد تطبيقها.

الخاصية في الصورة أعلاة color هي الخاصية المسؤولة عن تغيير لون الخط.
أنت غير ملزم بحفظ هذه الخصائص فسوف تنساها وإنما مع الوقت والممارسة ستحفظها ودائماً بإمكانك الرجوع للمرجع الخاص بها لتعلمها ومعرفة فائدتها وطريقة إستخدامها
إضغط هنا للذهاب لمرجع خصائص الـ CSS

- لاحظ طريقة كتابة الخاصية كالتالي color:

2- الـ Property Value أو قيمة الخاصية :

الخطوة التالية بعد أن تقوم بتحديد العنصر المستهدف والخاصية المطلوب تنفيذها هي أن تقوم بتحديد قيمة هذه الخاصية.

الصورة أعلاة توضح قيمة خاصية تغيير لون الخط حيث قمت بطلب تغيير لونة إلى اللون الأحمر red

- لاحظ هناك عدة طرق لتحديد اللون المطلوب بإمكانك تعلمها من خلال هذا المرجع وسنتطرق لها لاحقاً.

ملاحظات مهمة:

  • لاتنسى إغلاق كل خاصية بعد كتابة قيمتها بالفاصلة المنقوطة ; حتى وإن كانت الخاصية الأخيرة حتى تعتاد على ذلك وتتجنب الأخطاء.

هناك معايير محددة لعمل تسميات لأسماء الـ id و الـ class لجعل كودك مرتب ونظيف أحرص على التالي:

1- قم بإختيار أسماء منطقية حيث بمجرد مراجعتك لكود CSS تعرف اين هو العنصر المقصود بالتنسيق المحدد

على السبيل المثال: اذا لديك قسم في الموقع خاص بالعملاء قم بتسمية clients وليس sdfgd

<div id="clients"></div>

2- اذا اردت كتابة أكثر من وصف لإسم القسم قم بفصلها بعلامة -

على سبيل المثال تريد عمل قسم لـ clients feedback لا تقوم بوضعة بالمسافة هكذا

<div id="clients feedback"></div>

فهذا الكود يعني أنك أعطيته أثنين كلاسات وليس واحد

قم بعمل التالي لجعلة كلاس واحد:

<div id="clients-feedback"></div>

3- قم بكتابة أسماء الـ class والـ id بأحرف صغيرة

على سبيل المثال لا تقم بالتالي:

<div id="Clients"></div>

بل قم بالتالي:

<div id="clients"></div>

في الختام تذكر أن الـ CSS من أجمل مميزاتها أنك تقوم بكتابة الكود مرة واحدة وتطبيقة على أكثر من عنصر فلا تتجاهل هذا الميزة وتقوم بتكرار اكوادك

لنفترض أنك أردت تطبيق نفس التنسيق على كلاسين مختلفين فلا تقم بالتالي:

.paragraph{
  color:red;
}
.client-name{
  color:red;
}

لاحظ نريد تطبيق نفس التنسيق بنفس اللون فبإمكاننا دمج هذا الكود بالشكل التالي:

.paragraph, .client-name{
  color:red;
}

فقط قم بفصل كل كلاس عن الأخر بفاصلة ودعهم يتشاركوا نفس الخصائص
