إستنساخ cloning

عملية إستنساخ مشروع يوجد على أي سيرفر أخر أو على جهازك تسمى بالـ cloning. وتحتاج للقيام بهذه العملية للحصول على نسخة مطابقة من المشروع حتى نبدأ بالعمل علية.

العمل على واجهة الأوامر CLI:

للقيام بذلك كل ماعليك فعلة هو كتابة الأمر التالي:

git clone <url> <name>

حيث أن:

الـ <url> موقع المستودع سوى كان على سيرفر أو على جهازك

الـ <name> إسم المجلد الذي سيتم عمل الإستنساخ إلية. في حال تم كتابة إسم مجلد غير موجود سيتم إنشاء ذلك المجلد ومن ثم القيام بعملية الإستنساخ

ملاحظة: تحديد إسم المجلد إختياري وبإمكانك عمل الإستنساخ بالأمر التالي فقط

git clone <url>

إستنساخ مستودع محلي (Local) يتم كالتالي:

سأقوم بكتابة أمر الإستنساخ متبوع بمسار المجلد الخاص بالمستودع ومن ثم متبوع بإسم المجلد الذي أريد أن يتم عمل الإستنساخ إلية.

git clone webpage test

شاهد الخطوات:

لاحظ أن webpage ليس إسم المجلد فقط بل هو مسار المجلد وقمت بكتابة إسمة فقط لأنني قمت بفتح نافذة الـ CLI في نفس موقع المستودع, لنفترض بأنني أريد أن أقوم بعمل إستنساخ للمشروع وانا في مسار مختلف حينها سأقوم بكتابة مسار ملف المستودع على جهازي.

إستنساخ مستودع من على سيرفر (Remote) يتم كالتالي:

سأقوم بكتابة أمر الإستنساخ متبوع بعنوان المستودع على السيرفر

git clone https://github.com/FortAwesome/Font-Awesome

شاهد الخطوات:

لاحظ أنني هذه المرة لم أقم بتحديد إسم الملف الذي سيتم عمل الإستنساخ بداخله لهذا سيتم إنشاء ملف بشكل تلقائي يحمل نفس إسم المشروع على الـ GitHub. ولهذا تم إنشاء المجلد بإسم Font-Awesome.

مهمة اليوم

سنقوم بعمل تطبيق على بعض الأوامر ونعمل كفريق عن طريق القيام بلعبة سؤال وجواب .

قمت بعمل الصفحة أعلاة ورفعتها على مستودع, المهمة المطلوبة منك في هذا الدرس كالتالي:

1- عمل clone للمشروع على جهازك من هنا رابط المستودع.

2- بعد قيامك بذلك قم بمشاركة حسابك على موقع GitHub حتى يتم إضافتك كمساهم في المشروع.
